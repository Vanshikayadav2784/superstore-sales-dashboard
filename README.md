# 📊 Superstore Sales Dashboard

Welcome to my first data analysis project! 🚀  
This project explores sales performance using Excel and turns raw data into meaningful insights.

---

## 🧠 Project Objective
To analyze Superstore sales data and answer key business questions like:
- Which region performs best?
- Which category is most profitable?
- How are sales changing over time?

---

## 📁 Dataset
- Source: Superstore Sales Dataset (Kaggle)
- Data includes:
  - Order Date
  - Sales
  - Profit
  - Region
  - Category

---

## 🔍 Key Insights

✨ **Top Performing Region**  
- West region generated the highest sales

💰 **Most Profitable Category**  
- Technology category produced the highest profit

📈 **Sales Trend**  
- Sales show an overall increasing trend over time
