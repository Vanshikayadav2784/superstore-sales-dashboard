# Superstore Sales Dashboard

## 📊 Project Overview
This project analyzes Superstore sales data using Excel.

## 🔍 Key Insights
- West region has highest sales
- Technology category gives highest profit
- Sales trend shows growth over time

## 🛠 Tools Used
- Microsoft Excel

## 📁 Dataset
Superstore Sales Dataset from Kaggle

## 📌 What I Did
- Created Pivot Tables
- Analyzed sales by region and category
- Built a dashboard using charts

## 📷 Dashboard Preview
Dashboard created in Excel using pivot tables and charts.
